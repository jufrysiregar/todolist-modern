# To Do List Modern - Peppermint Style

Aplikasi To-Do List berbasis web yang dirancang dengan estetika modern, menggunakan efek **Glassmorphism**, animasi **Shatter** saat menghapus tugas, dan desain responsif untuk penggunaan mobile maupun desktop.

## ✨ Fitur Utama
- **Modern UI/UX**: Menggunakan Tailwind CSS untuk tata letak yang bersih dan animasi CSS kustom.
- **Group Management**: Kelola tugas berdasarkan kategori atau nama proyek.
- **Local Storage**: Data tersimpan secara otomatis di browser sehingga tidak hilang saat refresh.
- **Dynamic Animations**: Efek *shatter* (hancur) yang dramatis saat menghapus kegiatan.
- **Responsive Design**: Tampilan optimal di berbagai ukuran layar (HP, Tablet, PC).
- **Auto-height Textarea**: Input tugas yang otomatis memanjang mengikuti jumlah teks.

## 🚀 Cara Penggunaan
1. Unduh atau clone repositori ini.
2. Pastikan file `index.html`, `style.css`, dan `script.js` berada dalam satu folder yang sama.
3. Buka file `index.html` menggunakan browser pilihan Anda.
4. Mulai tambahkan kegiatan dengan menekan tombol **"+"** di pojok kanan atas.

## 📁 Struktur File
- `index.html`: Struktur utama aplikasi dan modal.
- `style.css`: Berisi semua desain animasi, gradient, dan styling manual.
- `script.js`: Menangani logika penyimpanan, render data, dan interaksi pengguna.
